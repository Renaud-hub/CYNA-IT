# CYNA‑IT — Documentation & Sécurité

Ce dépôt contient l’ensemble des documents, rapports et procédures liés au projet CYNA‑IT.  
Il regroupe les éléments de sécurité, les analyses SCA, les tests techniques ainsi que les runbooks d’exploitation.

---

## 📁 Structure du dépôt

### `docs/`
Dossier principal contenant la documentation du projet.

#### `docs/security-policy.pdf`
Politique de sécurité du projet CYNA‑IT :
- exigences de sécurité
- règles d’accès
- gestion des risques
- conformité

#### `docs/sca-report.pdf`
Rapport SCA (Software Composition Analysis) généré via Wazuh :
- dépendances détectées
- vulnérabilités identifiées
- niveaux de sévérité
- recommandations

#### `docs/tests/`
Rapports techniques :
- tests Nmap
- scans OWASP ZAP
- résultats d’analyse réseau
- preuves techniques pour le rapport final

#### `docs/runbooks/`
Procédures d’exploitation :
- gestion des incidents
- traitement des alertes Wazuh
- investigation Suricata
- procédures de sauvegarde / restauration

---

## 🎯 Objectifs du dépôt

- Centraliser toute la documentation du projet CYNA‑IT  
- Fournir un espace clair pour les rapports techniques  
- Mettre à disposition les runbooks pour l’exploitation  
- Faciliter la relecture et la validation du projet
